## 轉成 GitHub Page 遇到的問題：

使用 React Router 並設 basename={process.env.PUBLIC_URL}
故所有 Route 的 pathname 前端會有一串 "/myreact_gp"

- Public 圖片位址 (因為是首頁，img 的 src 可以使用 "./img/home/....")
